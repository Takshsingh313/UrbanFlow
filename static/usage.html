How to Use UrbanFlow

A quick start guide to operating the simulation and interpreting real-time data.

1

Start the Simulation

Navigate to the Simulation tab. The canvas will initialize empty. Use the Start control in the top right to begin the traffic flow.

2

Adjust Parameters

On the Configuration page, you can modify:

  • Car Density: Initial number of vehicles.
  • Slow Probability: Chance of random braking (0.0 - 1.0).
  • Lane Speed: Maximum velocity caps.
3

Analyze Telemetry

Switch to the Analytics tab to view real-time graphs:

  • Flow Rate: Cars per minute passing a point.
  • Avg Velocity: System-wide speed trends.
