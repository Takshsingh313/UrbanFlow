Nagel-Schreckenberg Model

A theoretical definition for the stochastic cellular automata used in this simulation.

Abstract

The Nagel-Schreckenberg (NaSch) model is a discrete model for traffic flow simulation. Its capability to reproduce traffic jams without external bottlenecks makes it a valuable tool in urban dynamics research.

Ruleset

For each vehicle $i$ at position $x$, velocity $v$, and gap $d$ to the next vehicle, the following rules update state $t \to t+1$:

  • 1. Acceleration $v \leftarrow \min(v + 1, v_{max})$
    Drivers want to reach the speed limit.
  • 2. Deceleration (Safety) $v \leftarrow \min(v, d)$
    Drivers brake to avoid collision.
  • 3. Randomization With probability $p$, $v \leftarrow \max(v - 1, 0)$
    Models distraction or over-braking.
  • 4. Update Position $x \leftarrow x + v$
